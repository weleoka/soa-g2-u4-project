Verify or change schedule ()
Undervisande lärare söker fram ett schema för en kurs i TimeEdit och kontrollerar att schemat överensstämmer med inskickat underlag samt att det inte finns dubbelbokningar med annat (ex. möten, uppdrag etc.) i sin kalender. Om det finns önskemål om ändringar i kursens schema kontaktar läraren den kursansvarige som vidarebefordrar förslag på ändringar via mejl till schemaläggningen som genomför ändringar i TimeEdit. Schemaläggaren meddelar via mejl till kursansvarig när ändringar i schemat är genomförda.
