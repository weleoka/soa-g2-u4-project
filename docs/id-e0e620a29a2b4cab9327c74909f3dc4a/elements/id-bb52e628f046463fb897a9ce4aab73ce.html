Implement schedule in LMS ()
När schemat är fastställt söker den kursansvarige fram schemat för en kurs i TimeEdit som underlag för manuell registrering av varje pass i canvaskalendern för kursen. I samband med denna registrering skrivs information om titel: ex. kurskod, lektion/handledning, plats: lokal + länk till Zoom-rum samt vem som är lärare och kortfattat om passets innehåll.
